# Library Mangement System
### A desktop application for library management.

### Description

Library Management System is a desktop 
application built using Java , JavaFx and MySQL database.

### Key Features

User

1. User registration
2. User log in
3. Search Book
4. Update user profile.
5. Request Book.
6. Borrow Book.


Admin
1. Admin Login
2. New Admin adding.
3. Add Book.
4. Issue book.
5. Delete Book.
6. Update Book Info.


### Tecnology Used

1. Java.
2. JavaFX.
3. MySQL
